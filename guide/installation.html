インストール方法 ​

配布されている zip ファイルを展開すると、MugTypography.ofx.bundle フォルダとインストーラースクリプトが入っています。スクリプトをダブルクリックするだけでインストールが完了します。

  1. DaVinci Resolve が起動している場合は終了させます。

  2. zip ファイルを展開します。

  3. install-win.bat をダブルクリックします。発行元を確認できない旨のセキュリティ警告が表示されるので、 「実行」 をクリックします。

  4. 管理者権限の確認ダイアログが表示されたら 「はい」 を選択します。

  5. インストール完了のメッセージが表示されたら、DaVinci Resolve を起動します。

  1. DaVinci Resolve が起動している場合は終了させます。

  2. zip ファイルを展開します。

  3. install-mac.command をダブルクリックします。macOS の Gatekeeper により「開いていません」というダイアログが表示されます。**「完了」**を押して閉じてください。

  4. システム設定 → プライバシーとセキュリティを開きます。下部に「"install-mac.command" がブロックされました」という表示と 「このまま開く」 ボタンが表示されるので、それをクリックします。

  5. 今度は「開きますか？」という確認ダイアログが表示されます。 「このまま開く」 をクリックします。

  6. Terminal に Password: と表示されたら、Mac のログインパスワードを入力して Return キーを押します。入力中は文字や記号が表示されませんが、正常な動作です。

  7. インストール完了のメッセージが表示されたら、DaVinci Resolve を起動します。

インストールの確認 ​

DaVinci Resolve のメニューから DaVinci Resolve → Preferences → Video Plugins を開き、MugTypography.ofx.bundle のステータスが Loaded successfully になっていれば完了です。
