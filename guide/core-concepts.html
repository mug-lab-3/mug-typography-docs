基本的な考え方 ​

このプラグインには、操作の対象となる 3 つのレベル があります。

レベル説明例
グローバルテキスト全体に一括で適用される設定フォント、全体の位置・回転・スケール
キャラクター1 文字ごとに個別に適用できる設定特定の文字を大きくする、色を変える
パーツ文字を構成するパス要素ごとの設定「だ」の点だけを動かす

たとえば、グローバルで全体を右に移動しつつ、特定の文字だけをさらに上にずらし、その文字の中の特定パーツだけを回転させる、といった重ね合わせが可能です。

設定値の伝播と合成 ​

各レベルの設定値は 上位から下位へ順に伝播し、すべてが合成された結果 として描画されます。

MGlobal回転 +10°テキスト全体のピボットを中心に
×
MCharacter回転 +15°その結果へ、文字のピボットを中心に
×
MPart回転 +20°さらに、パーツのピボットを中心に
＝
MFinal向き +45°向きは合計45°になる一方、描画位置は各ピボットの位置関係で決まり、単純な足し合わせにはならない
  • 太陽 ＝ Global（テキスト全体）
  • 地球 ＝ Character（文字ごと）
  • 月 ＝ Part（パーツごと）

地球は太陽のまわりを回り、月はさらに地球のまわりを回ります。それぞれは自分の軸を中心に素直に回っているだけでも、太陽から見た月の位置と向きは、すべての回転が掛け合わさった複雑な軌跡を描きます。Global・Character・Part の変形もこれと同じ仕組みで、各レベルのピボットを中心とした変形が順に掛け合わさって最終的な描画位置・向きが決まります。

Global → Character → Part の順に変形が積み重なる様子

色の合成についても同様の優先順位があります。詳細は付録の「色の優先順位」を参照してください。

パーツ操作が不要な場合 ​

パーツ単位での操作（移動・拡大縮小など）を使わない場合は、Lite版を選択することでクリップのロードや選択動作を軽くできます。Part Settings がインスペクタから省略され、項目数が減るためです。

目的に合ったプリセットを選ぶことが重要です。詳しくは「使い方・クイックスタート」を参照してください。

インデックスの見方 ​

Character Settings の Character 1, 2... は、入力テキストの先頭から数えた文字番号です。Part Settings の Part Setting 1, 2... は、全文字のパーツを通した番号です。これらは Output Settings や Animation Settings の Targets で使う c1、p1 と対応します。

TIP

Inspector の各項目には実際の文字と、その文字内で何番目のパーツかが表示されます。番号だけで判断しづらい場合は、ビューアで対象を選択して対応する項目を確認してください。
