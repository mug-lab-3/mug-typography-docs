使い方 ​

本プラグインには、2D用の6プリセットと3D用の2プリセットがあります。まず平面的なタイポグラフィか、カメラと奥行きを使う3D表現かを選び、次にパーツ編集の有無とテキスト量に合わせてプリセットを決めます。

Trial版では、選択したプリセットに関わらず最大文字数が一律 8文字に制限されます。

STEP 1: 2Dか3Dかを選ぶ ​

基本

Mug Typography 2D

通常のタイトル・字幕・モーションに

位置、回転、スケール、ストレッチを中心に、文字とパーツを平面上で編集します。Low / Mid / Highから処理上限を選べます。

Mug Typography 3D

カメラ・遠近感・奥行きを使う表現に

Camera Settingsと3D変形を搭載。文字やパーツをZ方向へ配置できます。上限はMid相当です。詳しくは「3Dタイポグラフィを作る」を参照してください。

STEP 2: 通常版か Lite 版かを選ぶ ​

Mug Typography 通常版

パーツ単位で細かく制御したい方に

Part Settings（パーツ個別調整）を搭載。文字を構成するパーツ一つひとつにパラメータを設定できます。

Mug Typography Lite 版

パーツ単位の編集が不要な方に

Part Settings を省略し、ロードと処理が高速。最大文字数は通常版の 1.5倍、パーツ数は描画上限の 512 まで扱えます。

STEP 3: 2Dではサイズ（Low / Mid / High）を選ぶ ​

3D版はMid相当の上限に固定されています。2D版を選んだ場合は、入力するテキストの長さや文字の複雑さに合わせてサイズを選びます。

サイズ Low

短いテキストに

16Lite: 24最大文字数
60Lite: 512最大パーツ数

動作が最も軽快です

推奨

サイズ Mid

通常のテキストに

30Lite: 45最大文字数
84Lite: 512最大パーツ数

迷ったらまずこちらを選択

サイズ High

長文・複雑な漢字に

60Lite: 90最大文字数
170Lite: 512最大パーツ数

⚠️ メモリ消費が大きめです

2Dの6種類と3Dの2種類を合わせ、使用できるプリセットは計8種類です。

DaVinci Resolve のエフェクトライブラリでの表示

TIP

パラメーター数が多いプリセットほど、クリップやノードを切り替えた際のレスポンスが低下します。必要な文字数・パーツ数を考慮して、適切なプリセットを選択してください。 各プリセットの上限値は、preferences.json の maxCharacters / maxParts で変更することもできます（調整方法は付録の「環境設定のカスタマイズ (preferences.json)」を参照）。

NOTE

各上限を超えた場合、超えた分は描画されません。

  • 最大文字数を超えた場合: 先頭から数えて上限を超えた文字以降が描画されません。
  • 最大パーツ数の上限に達した場合: 上限に達した時点の文字以降が、文字単位でまとめて描画されなくなります（1文字の一部だけが欠けることはありません）。

現在有効な上限値は、Inspector 最上部の Version 欄に (C: 文字数, P: パーツ数) の形式で常時表示されています。テキストが表示されない場合は、まずここを確認してください。

Edit ページ ​

エフェクトライブラリの Open FX には、用途に合わせて以下の2つの場所に上記の8プリセットが用意されています。タイムラインに配置またはクリップへ適用してご使用ください。

Generator

  • Effect Library
  • Open FX
  • Generators
  • MugLab
  • Mug Typography (Low / Mid / High / Lite / 3D…) 全8種

新規クリップとして配置

タイムライン上に新規クリップとして直接配置したい場合に使用します。ドラッグ＆ドロップでタイムライン上に配置できます。

Filter

  • Effect Library
  • Open FX
  • Filters
  • MugLab
  • Mug Typography (Low / Mid / High / Lite / 3D…) 全8種

既存クリップへのエフェクトとして適用

既存のクリップに対して、エフェクトとして本プラグインを適用したい場合に使用します。Color ページなどの他のページではこちら（Filters）のみが表示されます。

ビューア上での操作を有効にする ​

プレビュー画面上でのハンドル操作（移動・回転・スケールなど）を使うには、タイムライン左下の四角く囲った部分をクリック → Open FX Overlay に設定してください。

Edit ページでの使用例（左下の切り替えで Open FX Overlay を選択）

NOTE

Edit ページには DaVinci Resolve の仕様による制限があります。詳細は付録の「制限事項」を参照してください。

Fusion ページ ​

Fusion ページでは、以下の場所から必要なプリセットのノードをノードグラフに追加して使用します。 Edit ページと同じ8種類のプリセットが、すべてノードとして追加できます。

  • Tools
  • OpenFX
  • MugLab
  • Mug Typography (Low / Mid / High / Lite / 3D…) 全8種

ノードの Input 端子は、接続しなくても単体で使用できます。接続した場合は、入力画像が背景として描画され、その上にテキストが合成されます。
