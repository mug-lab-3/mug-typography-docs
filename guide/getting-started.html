使い方 ​

本プラグインのプリセットは2段階で選択します。まず、文字を構成するパーツ単位で制御したいかどうかで通常版／Lite 版を選び、次に入力するテキストの長さや文字の複雑さに合わせて サイズ（Low / Mid / High） を選びます。

Trial版では、選択したプリセットに関わらず最大文字数が一律 8文字に制限されます。

STEP 1: 通常版か Lite 版かを選ぶ ​

Mug Typography 通常版

パーツ単位で細かく制御したい方に

Part Settings（パーツ個別調整）を搭載。文字を構成するパーツ一つひとつにパラメータを設定できます。

Mug Typography Lite 版

パーツ単位の編集が不要な方に

Part Settings を省略し、ロードと処理が高速。最大文字数は通常版の 1.5倍、パーツ数は描画上限の 512 まで扱えます。

STEP 2: サイズ（Low / Mid / High）を選ぶ ​

サイズ Low

短いテキストに

16Lite: 24最大文字数
60Lite: 512最大パーツ数

動作が最も軽快です

推奨

サイズ Mid

通常のテキストに

30Lite: 45最大文字数
84Lite: 512最大パーツ数

迷ったらまずこちらを選択

サイズ High

長文・複雑な漢字に

60Lite: 90最大文字数
170Lite: 512最大パーツ数

⚠️ メモリ消費が大きめです

2つの選択の組み合わせで、使用するプリセットは計6種類の中から決まります。

DaVinci Resolve のエフェクトライブラリでの表示

NOTE

各上限を超えた場合、超えた分は描画されません。

  • 最大文字数を超えた場合: 先頭から数えて上限を超えた文字以降が描画されません。
  • 最大パーツ数の上限に達した場合: 上限に達した時点の文字以降が、文字単位でまとめて描画されなくなります（1文字の一部だけが欠けることはありません）。

現在有効な上限値は、Inspector 最上部の Version 欄に (C: 文字数, P: パーツ数) の形式で常時表示されています。テキストが表示されない場合は、まずここを確認してください。

IMPORTANT

通常は Low もしくは Mid (またはそれらの Lite 版) を使用し、文字数・パーツ数が足りない場合にのみ High / LiteHigh を使用してください。 High は制限値が大きいぶんメモリを多く消費するため、搭載メモリの少ない環境では DaVinci Resolve ごとクラッシュする恐れがあります。 もし High / LiteHigh でクラッシュする場合は、preferences.json で High の maxCharacters / maxParts を実際に必要な値まで下げることで、安定して使用できる場合があります（調整方法は付録の「環境設定のカスタマイズ (preferences.json)」を参照）。

Edit ページ ​

エフェクトライブラリの Open FX には、用途に合わせて以下の2つの場所に上記の各プリセット（Low / Mid / High / Lite Low / Lite Mid / Lite High）が用意されています。タイムラインに配置または適用してご使用ください。

Generator

  • Effect Library
  • Open FX
  • Generators
  • MugLab
  • Mug Typography (Low / Mid / High / Lite…) 全6種

新規クリップとして配置

タイムライン上に新規クリップとして直接配置したい場合に使用します。ドラッグ＆ドロップでタイムライン上に配置できます。

Filter

  • Effect Library
  • Open FX
  • Filters
  • MugLab
  • Mug Typography (Low / Mid / High / Lite…) 全6種

既存クリップへのエフェクトとして適用

既存のクリップに対して、エフェクトとして本プラグインを適用したい場合に使用します。Color ページなどの他のページではこちら（Filters）のみが表示されます。

ビューア上での操作を有効にする ​

プレビュー画面上でのハンドル操作（移動・回転・スケールなど）を使うには、タイムライン左下の四角く囲った部分を クリック → Open FX Overlay に設定してください。

Edit ページでの使用例（左下の切り替えで Open FX Overlay を選択）

NOTE

Edit ページには DaVinci Resolve の仕様による制限があります。詳細は付録の「制限事項」を参照してください。

Fusion ページ ​

Fusion ページでは、以下の場所から必要なプリセットのノードをノードグラフに追加して使用します。 Edit ページと同じ6種類のプリセットが、すべてノードとして追加できます。

  • Tools
  • OpenFX
  • MugLab
  • Mug Typography (Low / Mid / High / Lite…) 全6種

ノードの Input 端子への接続有無によって、動作が切り替わります。Input を接続しなくてもジェネレーターとして単体で使用できます。

  • Input 未接続: ジェネレーターとして動作
  • Input 接続あり: フィルターとして動作
