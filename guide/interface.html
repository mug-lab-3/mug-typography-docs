インターフェース概要 ​

選択モード（Selection Mode） ​

「基本的な考え方」で説明した3つの階層（Global / Character / Part）を切り替えます。3DプリセットとScriptingプリセットでは、カメラ操作用のCameraモードも選択できます。ビューア上でのクリックやドラッグは、ここで選んだモードに応じて動作します。

パラメーター操作対象レベル説明デフォルトショートカット
Globalグローバルテキスト全体を選択対象にするWinAlt+Ctrl+A/MacOption+Command+A
Characterキャラクター個々の文字を選択対象にするWinAlt+Ctrl+C/MacOption+Command+C
Partパーツ文字内のパス要素を選択対象にするWinAlt+Ctrl+P/MacOption+Command+PLite / Scriptingでは無効
Cameraカメラ画面上に専用の操作ハンドルを表示し、カメラを操作するWinAlt+Ctrl+M/MacOption+Command+M3D / Scriptingのみ

Inspector 最上部の Selection Mode から切り替えます。

ScriptingプリセットではCharacter / Partモードが表示されず、Global / Cameraのみ選択できます。文字・パーツの個別値はLuaスクリプトから制御します。

Inspector 最上部の Selection Mode（ここで操作対象の階層を切り替える）

NOTE

Output Settings で描画対象外になっている文字・パーツは、クリックやマーキーで選択することもできません。

ビューア画面での操作 ​

ビューア上では、対象を直接ドラッグするほか、修飾キーや操作ハンドルを使って変形・選択できます。

NOTE

文字の周りに縁取りのような線が表示されますが、これは文字がバウンディングボックス外に出た場合でも選択・操作できるようにするためのインタラクション用オーバーレイ表示です。表示のON/OFFや色はOverlay Settingsで変更できます。

移動 ​

文字やパーツの上でクリック＆ドラッグすると、位置を移動できます。

  • すでに選択済みの対象をクリックすると、複数選択を維持したまままとめて移動できます。
  • 別の対象をクリックすると、そちらに選択が切り替わります。

文字をクリック＆ドラッグで移動する様子（選択済みの複数文字はまとめて移動）

複数選択の追加・解除 ​

Character または Part モードで、WinCtrl+Alt/MacCommand+Option を押しながら対象をクリックすると、その文字・パーツの選択状態を切り替えられます。

  • 未選択の対象をクリックすると、現在の選択に追加されます。
  • 選択済みの対象をクリックすると、現在の選択から解除されます。
  • 空白部分をクリックしても、現在の選択は変わりません。

Ctrl+Alt（Command+Option）+ クリックで文字の選択を追加・解除する様子

回転・スケール・ピボットハンドルの表示 ​

選択済みの対象の上で WinCtrl/MacCommand を押しながらクリックすると、回転・スケール・ピボット操作用のハンドルが表示されます。

ハンドル位置操作
回転ハンドルピボットから伸びた先の丸左右へドラッグで回転
スケールハンドルピボットから伸びた先の長方形前後へドラッグでスケール変更
ピボットハンドル中心の十字マーク（回転・スケールハンドルの根元）ドラッグで回転・スケールの基準点を移動

選択済みの対象を Ctrl（Command）+ クリックすると表示される回転・スケール・ピボットハンドル

NOTE

選択済みの対象の上で WinCtrl/MacCommand + クリックしたときのみハンドルが表示されます。未選択の対象では表示されません。

ストレッチ ​

WinAlt/MacOption を押しながら、バウンディングボックスの辺（上・下・左・右）をドラッグすると、その方向の Stretch を変更できます。選択中の対象に対して作用します。

Alt（Option）を押しながらバウンディングボックスの辺をドラッグして Stretch を変更する様子

Tracking / マージンの調整 ​

WinAlt+↑/↓/MacOption+↑/↓ を押すと、0.1em 単位で値を増減できます。
モード適用対象
GlobalTracking（グローバル文字間隔）
Character選択中の文字の Margin After
Part使用不可
WinAlt/MacOption を押している間は、プレビュー画面に各文字の Tracking および Margin After の範囲が表示されます。いま調整中の対象（Global モードなら Tracking、Character モードなら Margin After）が Previewing Color（プレビュー色）で強調されます。

Alt（Option）押下中の Tracking プレビュー表示（Global モード）

Alt（Option）押下中の Margin After プレビュー表示（Character モード）

マーキー選択（範囲選択） ​

文字やパーツのない空白エリアでドラッグすると、選択矩形が表示され、矩形内に含まれる対象をまとめて選択できます。

ドラッグ中、選択矩形の範囲内に入っている要素は Selected Color（選択色）でリアルタイムにハイライト表示され、範囲外の要素は Previewing Color（プレビュー色）で表示されます。

マーキー選択の様子（矩形の範囲内は選択色、範囲外はプレビュー色で表示）

Cameraモード（3D / Scriptingプリセット） ​

Mug Typography 3DまたはMug Typography Lite (3D)では、Selection ModeにCameraが追加されます。選択するとビューア上に専用の操作ハンドルが表示され、Inspectorを往復せずに構図を調整できます。

Cameraモードの操作ハンドル（Orbit / Target Pad / Zoom / Perspective）

操作動作
OrbitハンドルをドラッグカメラのYaw/Pitchを変更
空白部分を WinAlt/MacOption + ドラッグカメラをオービット
Target Padをドラッグカメラの注視点を移動
Zoomハンドルを左右へドラッグCamera Zoomを変更
Perspectiveハンドルを左右へドラッグ遠近感の強さを変更
操作ハンドル以外の空白部分をドラッグLens Shiftを変更
ドラッグ中にShift移動量の大きい軸へ固定

各設定の意味はCamera Settingsを参照してください。
