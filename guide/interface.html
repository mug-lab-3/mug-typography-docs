インターフェース概要 ​

選択モード（Selection Mode） ​

「基本的な考え方」で説明した 3 つのレベル（グローバル・キャラクター・パーツ）のどれを操作対象にするかを切り替えます。プレビュー画面上でのクリックやドラッグ操作は、ここで選んだレベルに対して作用します。

パラメーター操作対象レベル説明デフォルトショートカット
Globalグローバルテキスト全体を選択対象にするAlt+Ctrl+A
(Mac: Option+Command+A)
Characterキャラクター個々の文字を選択対象にするAlt+Ctrl+C
(Mac: Option+Command+C)
Partパーツ文字内のパス要素を選択対象にするAlt+Ctrl+P
(Mac: Option+Command+P ※Lite版は無効)

NOTE

後述の Output Settings で描画対象外になっている文字・パーツは、クリックやマーキーで選択することもできません。

プレビュー画面での操作 ​

プレビュー画面上での操作は、修飾キーの組み合わせによって以下の 3 つのモードに分かれます。

移動 ​

文字やパーツの上でクリック＆ドラッグすると、位置を移動できます。

  • すでに選択済みの対象をクリックすると、複数選択を維持したまままとめて移動できます。
  • 別の対象をクリックすると、そちらに選択が切り替わります。

回転・スケール・ピボットハンドルの表示 ​

選択済みの対象の上で Ctrl キー（macOS: Command キー）を押しながらクリックすると、回転・スケール・ピボット操作用のハンドルが表示されます。

ハンドル位置操作
回転ハンドルピボットから伸びた先の丸左右へドラッグで回転
スケールハンドルピボットから伸びた先の長方形前後へドラッグでスケール変更
ピボットハンドル中心の十字マーク（回転・スケールハンドルの根元）ドラッグで回転・スケールの基準点を移動

NOTE

選択済みの対象の上で Ctrl+クリックしたときのみハンドルが表示されます。未選択の対象では表示されません。

ストレッチ ​

Alt キー（macOS: Option キー）を押しながら、バウンディングボックスの辺（上・下・左・右）をドラッグすると、その方向の Stretch を変更できます。選択中の対象に対して作用します。

Tracking / マージンの調整 ​

Alt キー（macOS: Option キー）を押しながら ↑ / ↓ キーを押すと、0.1em 単位で値を増減できます。

モード適用対象
GlobalTracking（グローバル文字間隔）
Character選択中の文字の Margin After
Part使用不可

Alt キー（macOS: Option キー）を押している間は、プレビュー画面に各文字の Tracking および Margin After の範囲が表示されます。いま調整中の対象（Global モードなら Tracking、Character モードなら Margin After）が Previewing Color（プレビュー色）で強調され、もう一方はグレーで表示されます。これにより、どちらを調整しているかを見分けられます。

マーキー選択（範囲選択） ​

文字やパーツのない空白エリアでドラッグすると、選択矩形が表示され、矩形内に含まれる対象をまとめて選択できます。

ドラッグ中、選択矩形の範囲内に入っている要素は Selected Color（選択色）でリアルタイムにハイライト表示され、範囲外の要素は Previewing Color（プレビュー色）で表示されます。
