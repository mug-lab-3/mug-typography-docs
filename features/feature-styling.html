STYLE & LEGIBILITY

アクセントで、印象に残るデザインに

ストローク、ドロップシャドウ、背景ボックスをひとつのタイポグラフィへ。視認性を高め、より読みやすいテキストを作成できます。

  • 多彩なストローク
  • 文字別シャドウ
  • 背景ボックス
使い方を見る設定リファレンス

使い方 ​

  1. Page を Global Settings にします。
  2. Stroke を開き、Width を上げます。
  3. Color、Join Style、Draw Order を仕上がりに合わせて選びます。

Gap を上げると、塗りとストロークの間に透明な隙間を作れます。Transparent Fill と組み合わせれば、輪郭だけの中抜き文字になります。

設定見た目への影響
Join Style角を鋭く、丸く、面取り状に切り替える
Miter Limit鋭い角が長く伸びすぎるのを抑える
Draw Order塗りとストロークのどちらを前面にするか決める
Gap塗りとストロークの間隔を空ける

シャドウで奥行きをつける ​

  1. Page を Global Settings にします。
  2. Shadow Settings を開き、Draw Global Shadow をオンにします。
  3. Distance と Angle で影の位置を決めます。

特定の文字だけ影色を変える場合は Character Settings の Use Indiv Shadow Color を使います。この個別設定は、Draw Global Shadow がオフでも対象文字の影を有効にします。

背景ボックスを敷く ​

  1. Page を Global Settings にします。
  2. Bounding Box Settings を開き、Fill Enabled をオンにします。
  3. Offset と Padding で文字との位置や余白を調整します。
  4. 必要に応じて Stroke Enabled で枠線を追加します。
