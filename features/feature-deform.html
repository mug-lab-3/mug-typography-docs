MOTION & DEFORMATION

輪郭そのものが動きはじめる

パルス、台形、球面、樽型、手描き風。文字のアウトラインへ直接働く5種類の変形を重ね、有機的なモーションを生み出せます。

  • 5種類の組み込み効果
  • 文字ごとの位相制御
  • 対象パーツを限定
使い方を見る設定リファレンス

エフェクト一覧 ​

エフェクト動きサンプル
Pulse（パルス）輪郭を内外へ膨張・収縮
Trapezoid（台形）指定した軸に沿って台形状に変形
Sphere（球面）球面のように中央を膨張・収縮
Barrel（樽型）樽型・糸巻き型に歪ませる
Hand Drawn（手描き風）輪郭の揺れと姿勢のジッターを合成

TIP

Hand Drawn の Pattern Count を 1 に設定すると、揺れのパターンを固定できます。静止画として手描き感が欲しいときに便利です。

使い方 ​

  1. Page を Animation Settings にします。
  2. 使いたいエフェクトのグループ内の Enabled をオンにします。
  3. Amplitude（Hand Drawn では Wobble / Jitter）で変形量を決めます。
  4. Speed (Hz) と Phase で動く速さと開始位置を整えます。
  5. 必要なら Targets で適用先を限定します。

TIP

デフォームは複数同時に適用できます。効果を重ねると、複雑な動きを作れます。

文字ごとに動きのタイミングをずらす ​

Char Phase は文字ごとの動きの位相を、Char Delay は文字ごとの開始時刻をずらします。連続した文字へ順番に動きを伝えるなら Char Phase、時間差ではっきり始動させるなら Char Delay が向いています。

Step を上げると動きを連続値から段階的な変化へ近づけられます。

Targets で一部だけ変形する ​

変形エフェクト（デフォーム）は、特定のパラメータに決められたフォーマットで文字やパーツを指定することで、変形アニメーションの適用対象を一部に限定させることができます。

各エフェクトの設定グループ内にある Targets フィールドに記述します。 このフィールドが空欄の場合、エフェクトはテキスト全体のすべてのパーツに適用されます。限定する場合は、文字を c、パーツを p とし、それぞれ1から始まる番号を続けて指定します。

text
c1-c3       # 1〜3文字目
p2 p5       # 2番目と5番目のパーツ
!c1         # 1文字目以外

番号は Character Settings / Part Settings の並びと対応します。詳しい文法は Targets の記述フォーマット を参照してください。
