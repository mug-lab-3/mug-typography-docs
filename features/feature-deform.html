MOTION & DEFORMATION

輪郭そのものが動きはじめる

パルス、台形、球面、樽型、手描き風。文字のアウトラインへ直接働く5種類の変形を重ね、有機的なモーションを生み出せます。

  • 5種類の組み込み効果
  • 文字ごとの位相制御
  • 対象パーツを限定
使い方を見る設定リファレンス

エフェクトを選ぶ ​

エフェクト動きサンプル
Pulse輪郭を内外へ膨張・収縮
Trapezoid指定した軸に沿って台形状に変形
Sphere球面のように中央を膨張・収縮
Barrel樽型・糸巻き型に歪ませる
Hand Drawn輪郭の揺れと姿勢のジッターを合成

設定手順例 ​

  1. Page を Animation Settings にします。
  2. 使いたいエフェクトのグループを開き、Enabled をオンにします。
  3. Amplitude（Hand Drawn では Wobble / Jitter）で変形量を決めます。
  4. Speed (Hz) と Phase で動く速さと開始位置を整えます。
  5. 必要なら Targets で適用先を限定します。

TIP

デフォームは複数同時に適用できます、弱い効果を重ねると複雑な動きを作れます。

文字ごとに波をずらす ​

Char Phase は文字ごとの波の位相を、Char Delay は文字ごとの開始時刻をずらします。連続した文字へ順番に動きを伝えるなら Char Phase、時間差ではっきり始動させるなら Char Delay が向いています。

Step を上げると動きを連続値から段階的な変化へ近づけられます。

Targets で一部だけ変形する ​

Targets が空欄なら、エフェクトはすべてのパーツに適用されます。文字は c、パーツは p に1始まりの番号を続けて指定します。

text
c1-c3       # 1〜3文字目
p2 p5       # 2番目と5番目のパーツ
!c1         # 1文字目以外

番号は Character Settings / Part Settings の並びと対応します。詳しい文法は Targets の記述フォーマット を参照してください。

手描き風を静止させる ​

Hand Drawn の Pattern Count を 1 にすると、揺れのパターンを固定できます。静止画として手描き感が欲しいときに便利です。
