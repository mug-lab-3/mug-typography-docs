ライセンス

WRITE ON

文字を順番に表示させる

文字やパーツを、先頭から順に登場・退場させる演出。キーフレームを2つ置くだけで、手書きのように文字を書き進めるアニメーションを作れます。

  • キーフレーム2つで完成
  • 書き順風のパーツ出現
  • 出現順を自由に指定
使い方を見る設定リファレンス

使い方 ​

  1. Page を Output Settings にします。
  2. アニメーションの開始フレームで End を 0.0 にキーフレームします。
  3. 終了フレームで End を 1.0 にします。

これだけで、文字が先頭から順に現れるアニメーションが完成します。逆に Start をキーフレームすると、先頭から順に消えていく動きになります。

単位を選ぶ ​

Unit動作
Character文字単位で出現・消滅します。1文字ずつポンポンと現れる表現に向いています。
Partパーツ単位で出現・消滅します。Stroke Order と組み合わせると、書き順に近い順序でパーツが現れます。

出現順を自由に決める ​

Manual Order をONにし、Manual Order List へ文字・パーツを列挙すると、その順序で先に表示されます。

text
c4-c6, c9, c1-c3

上の例では4〜6文字目、9文字目、1〜3文字目の順に表示され、未指定の文字（7・8文字目）はその後に続きます。区切りには空白も使え、p2-p5 のようにパーツの範囲指定もできます。

詳しい仕様は「出力設定 (Output Settings)」を参照してください。
