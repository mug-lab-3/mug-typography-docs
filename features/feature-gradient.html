COLOR & GRADIENT

自由な組み合わせで美しく

Oklabなどの知覚色空間による自然な補間で、鮮やかな色同士も濁りにくく、好きな色どうしのグラデーションが行なえます。

  • 4種類の色空間
  • 自由な方向と範囲
  • 自然な中間色
使い方を見る設定リファレンス

最短手順 ​

  1. Page を Global Settings にします。
  2. Color Settings を開き、開始色を Font Color で決めます。
  3. Gradient を開き、Color SpaceをOklabなどNone以外にします。
  4. Gradient End Color で終端色を選びます。
  5. Angle で方向、Start / End で色が切り替わる範囲を整えます。

まずは Color Space を Oklab、Midpoint を 0.5 のまま試すと、自然な結果を得やすくなります。

Color Space の選び方 ​

Color Space補間結果向いている表現
Oklab明るさの変化が自然な、汎用的なグラデーション
Okhsl彩度と明度の感覚を保ちながら色相をつなぐ表現
Okhsv鮮やかさを保った色相変化
Linear RGB光の加算に近い補間や、従来型のRGB補間

TIP

中間の色が暗く見える場合は Oklab、色相の回り方を重視する場合は Okhsl または Okhsv を起点に調整してください。

範囲と中間点を整える ​

  • Angle: グラデーションの進行方向を回転します。
  • Start / End: 開始色と終端色に到達する位置を決めます。
  • Midpoint: 2色が釣り合う位置を開始側・終端側へ寄せます。
