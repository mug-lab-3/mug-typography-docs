OFFSET ANIMATION NEW

ひとつの動きを文字全体へ展開する

1文字に作成したキーフレームアニメーションを、ほかの文字やパーツへ時間差付きで複製。Frame Offsetを指定するだけで、文字が順番に動き出すFollowerのような連続アニメーションをすばやく作成できます。

  • 1つの動きから展開
  • Frame Offsetで時間差
  • 展開順を自由に指定
使い方を見る設定リファレンス

使い方 ​

  1. 元となる文字またはパーツ1つに、キーフレームアニメーションを作成します。
  2. Page を Copy / Paste にします。
  3. 手順1でアニメーションを作成した文字・パーツを選択し、Copy Selection を押します。
  4. Paste Target Attributes で、複製したいアニメーションの項目をONにします。
  5. プレビュー画面で、貼り付け先の文字・パーツを複数選択します。
  6. Frame Offset に、対象ごとにずらすフレーム数を入力します。
  7. Paste Animation with Offset を押します。

コピー元を基準（ずれなし）として、貼り付け先には Frame Offset の1倍、2倍…と累積した時間差でアニメーション全体が複製されます。Frame Offset を 3 にすると各対象が3フレームずつ遅れて動き出し、-3 にすると時間方向が逆になります。

IMPORTANT

貼り付け先の文字・パーツに設定済みだったアニメーションは、チェック対象のパラメーターについてすべて消え、コピー元の内容で置き換えられます。

Fusionページでの注意 ​

DaVinci ResolveのFusionページはOFXプラグインによるアニメーションの直接コピーに対応していないため、Editページより処理に時間がかかったり、余分なキーフレームが設定されたりする場合があります。

IMPORTANT

Fusionページでは、手順7のあとキーフレームの反映が落ち着くまで数秒待ち、選択やチェック状態を変えずに Repair Animation Paste を押してください。想定される時刻以外に付いたキーフレームが削除され、正しい時間差に整理されます。

展開する順序を指定する ​

デフォルトでは、論理インデックス順（c1、c2…の並び）に時間差が加算されます。Order に文字・パーツを列挙すると、その順序でオフセットを加算できます。

text
c3 c1 c2

上の例では3文字目が最初（Frame Offset × 1）、1文字目が次（Frame Offset × 2）…の順に動き出します。書式はWrite OnのManual Orderと同じで、p2-p5 のような範囲指定も使用できます。未指定の対象は論理インデックス順で末尾に続きます。

詳しい仕様（2Dパラメーターの軸の扱い、色チャンネルのキーフレーム条件など）は「コピー・ペースト (Copy / Paste)」を参照してください。
