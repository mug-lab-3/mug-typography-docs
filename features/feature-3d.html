3D TYPOGRAPHY

奥行き感のある、パースの効いた表現

テキスト全体を平面として傾けるだけでなく、文字やパーツを奥行き方向へ配置。専用カメラと遠近感を組み合わせ、2Dの操作感を保ったまま立体的なタイポグラフィを作れます。

  • ビューアでカメラ操作
  • 文字・パーツの奥行き
  • 既存エフェクトと併用
使い方を見るCamera Settings

使い方 ​

  1. エフェクトライブラリから Mug Typography 3D または Mug Typography Lite (3D) を配置します。
  2. Selection Mode を Camera に切り替えます。
  3. ビューアのOrbitハンドル、または空白部分のAltドラッグでカメラを回します。
  4. 必要に応じて Camera Settings の Perspective Strength と Camera Zoom で画角を仕上げます。

3D版とLite 3D版 ​

プリセット最大文字数パーツ設定向いている用途
Mug Typography 3D30最大84パーツパーツごとに奥行きを付ける立体表現
Mug Typography Lite (3D)45なし文字単位の3D表現、長めのテキスト

どちらも通常のMidプリセットを基準にした上限です。Lite 3DはPart Settingsを省略し、最大512パーツまで描画できますが、パーツ単位のZ Offsetは編集できません。

TIP

最大文字数・最大パーツ数の上限は、preferences.json の maxCharacters / maxParts で変更できます。調整方法は付録の「環境設定のカスタマイズ (preferences.json)」を参照してください。

3つの階層で立体感を作る ​

階層3D専用パラメーター役割
GlobalGlobal Yaw / Global Pitchテキスト全体の平面を傾ける
CharacterZ Offset / Yaw / Pitch文字ごとに奥行きと向きを変える
PartZ Offset点や画を前後へ分離する
CameraTarget / Yaw / Pitch / Perspective / Zoom / Lens Shiftシーンを見る位置と投影を決める

TIP

まずGlobal Yaw/Pitchで全体の向きを決め、次に文字のZ Offsetで前後差を作り、最後にカメラを調整すると迷いにくくなります。

2D版との違い ​

3Dプリセットでは、Character Settingsの Stretch が Z Offset / Yaw / Pitch に置き換わります。Part SettingsではStretchがZ Offsetに置き換わります。Position、Pivot、Scale、Rotationや、カラー・ストロークなどの既存設定は引き続き使用できます。

グラデーション、シャドウ、バウンディングボックス、デフォーム、モーフ、Write Onも3D投影後の見た目へ追従します。

NOTE

現在、Copy / Pasteページの「選択範囲をコピー」では、Z Offset / Yaw / Pitchを個別の貼り付け対象に選べません。

ビューア上での操作 ​

Selection Mode が Global / Character / Part のとき、テキストはカメラ設定を適用する前の状態で表示されます。移動・回転・スケールなどのビューア操作も、この状態の2D座標に対して行われます。カメラで付けた傾きや遠近感に惑わされることなく、2Dプリセットと同じ感覚でレイアウトを編集できます。

カメラを含めた最終的な構図を確認・調整するには、Selection Mode を Camera に切り替えます。
