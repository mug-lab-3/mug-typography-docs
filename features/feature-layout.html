文字・パーツを1つずつ動かす ​

文字単位、さらにはパーツ単位まで分解して、位置・回転・スケールを個別に調整できます。「特定の1文字だけ大きく傾ける」「『ら』の点だけ飛び出させる」といった細かい演出が可能です。

数値入力だけでなく、プレビュー画面上で対象の文字・パーツをクリックして選択し、表示されるハンドルをドラッグするだけで直感的に操作できます。

→ 詳しくは Character Settings / Part Settings
