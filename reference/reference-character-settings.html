PARAMETER REFERENCE

Character Settings

選択した文字へ、位置・回転・色・輪郭線・影を個別に追加します。複数文字の整列から、一文字単位のスタイリングまでをまとめたページです。

≡Align複数文字を水平・垂直整列↘↗Transform位置・ピボット・回転↘◐Color文字ごとの塗り色↘◇Stroke文字ごとの輪郭線↘◒Shadow文字ごとの影色↘

キャラクター選択モード（Character）で選択した文字に対して個別の変形を設定するページです。 パラメーターのスロット数は、選択しているプリセット（Low / Mid / High）や preferences.json の設定によって異なります（詳細は付録の「制限事項」を参照）。

キャラクターへの変形は、グローバル変形に追加で適用されます。

文字の整列（Align Selected） ​

ビューア上で複数選択した文字を、文字のスケールを考慮してきれいに並べ直すためのボタンです。インスペクタページの上部に配置されています。

IMPORTANT

選択モードの制限について
文字の整列機能は、ビューア上での文字選択モード（Character）が有効な場合のみ機能します。 グローバル（Global）およびパーツ（Part）選択モードでは機能しませんのでご注意ください。

パラメーター説明デフォルトショートカット
Align Selected Horizontally選択中の文字のうち、最も左にある文字の左端を基準として、他の文字を右方向へ隙間なく配置します。各文字のベースライン（垂直位置）が揃うように整列され、各文字のピボット（Pivot）はデフォルト（0.5, 0.5）にリセットされます。WinAlt+Ctrl+H/MacOption+Command+H
Align Selected Vertically選択中の文字のうち、最も上にある文字の上端を基準として、他の文字を下方向へ隙間なく配置します。列の中心軸（水平位置）が揃うように整列され、各文字のピボット（Pivot）はデフォルト（0.5, 0.5）にリセットされます。WinAlt+Ctrl+V/MacOption+Command+V

整列（Align Selected）サンプル ​

整列方向動作サンプル
水平整列 (Align Selected Horizontally)
垂直整列 (Align Selected Vertically)

NOTE

ショートカットキーに Alt + Ctrl を使用している理由
通常のショートカットキー（Alt + アルファベットなど）と異なり、Alt + Ctrl が割り当てられています。これは、DaVinci Resolve 自体が Alt + V を「属性の貼り付け (Paste Attributes)」などのホスト機能として優先消費してしまう仕様があり、プラグイン側でキー入力を確実にキャッチできるようにするための衝突回避策です。
※ショートカットの割り当てキーは、設定ファイル preferences.json でカスタマイズ可能です。

個々の文字の設定項目（Character 1, 2...） ​

「Character 1」などのグループを展開すると、選択した特定の文字に対する個別設定項目が表示されます。

リセット（Reset） ​

パラメーター説明
Reset This Characterその文字と、その文字に属するすべてのパーツの変形・余白・個別カラー・個別ストローク・個別シャドウカラーをデフォルト値に戻します。3Dプリセットでは、文字のZ Offset / Yaw / PitchとパーツのZ Offsetも対象です。

余白（Margin After） ​

パラメーター説明デフォルト
Margin After (em)その文字の後ろに追加するスペース量を em 単位で指定します。0.0

変形パラメーター ​

パラメーター説明デフォルト
Pos文字個別の位置オフセット（X, Y）。X0.5Y0.5
Pivot文字のスケール・回転の基準点（X, Y）。X0.5Y0.5
Scale文字を均等にスケールします。1.0
Rotation文字の回転角度。0°

2DプリセットでのStretch ​

パラメーター説明デフォルト
Stretch文字をX / Y方向に個別スケールします。3Dプリセットでは表示されません。X1.0Y1.0

3Dプリセットでの変形 NEW ​

3Dプリセットでは Stretchは表示されず、使用できません。

文字の奥行きと3D方向の傾きを調整するために、次のパラメーターが追加されます。

パラメーター説明デフォルト
Z Offset文字を奥行き方向へ移動します。キャンバス高さを基準とした値です。0.0
Yaw文字のPivotを中心に左右へ傾けます。正の値で右端がカメラから遠ざかります。0°
Pitch文字のPivotを中心に上下へ傾けます。正の値で上端がカメラから遠ざかります。0°

Position、Pivot、Scale、Rotationは3Dプリセットでも使用できます。カメラ全体の設定はCamera Settingsを参照してください。

個別の色 ​

パラメーター説明デフォルト
Use Indiv ColorON にすると、グローバルの色設定を上書きしてこの文字専用の色を使います。ON にすると Color が表示されます。OFF
Color文字ごとの色（RGB + 不透明度）。白#ffffff

個別の輪郭線（ストローク） ​

パラメーター説明デフォルト
Use Indiv StrokeON にすると、グローバルの輪郭線設定を上書きしてこの文字専用の輪郭線設定を使います。ON にすると、以下のパラメーターが表示されます。OFF
Stroke Width文字専用の輪郭線の太さを、フォントサイズに対する比率で指定します。0.0
Stroke Gap文字専用の、塗り（フィル）と輪郭線（ストローク）の間に空ける隙間の幅を、フォントサイズに対する比率で指定します。0.0
Stroke Color文字専用の輪郭線の色を指定します。オフブラック#1c1c1e

NOTE

Stroke Widthはフォントサイズを基準に計算され、文字のScaleやStretchには追従しません。

個別のシャドウカラー ​

パラメーター説明デフォルト
Use Indiv Shadow ColorON にすると、グローバルのシャドウカラー設定を上書きしてこの文字専用のシャドウカラー設定を使います。ON にすると、以下のパラメーターが表示されます。OFF
Shadow Color文字専用のシャドウの色（RGB + 不透明度）を指定します。Use Indiv Shadow Color が ON のときのみ有効。青#5555ff

NOTE

個別シャドウカラーを ON にした文字は、Global Settings の Draw Global Shadow が OFF でも影を落とします。特定の文字にだけシャドウを付けたい場合は、グローバルシャドウを OFF のままにして、対象の文字でこの Use Indiv Shadow Color を ON にしてください。 なお、シャドウの距離（Distance）・角度（Angle）は Global Settings の値が共通で使われます。個別に変えられるのは色のみです。
