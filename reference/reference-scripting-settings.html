PARAMETER REFERENCE

Scripting Settings

Lua 5.4スクリプトによるプロシージャルなアニメーション制御を設定します。

▤Inspectorプリセットごとの表示位置↘🎬Edit PageGenerator使用時の注意↘⚠️制限事項エラー時の挙動と実行制限↘

通常の2D / 3D / Liteプリセットでは、インスペクタの Page で Scripting を選択すると表示されるページです。専用のMug Typography Scriptingプリセットでは、同じ設定がGlobalページ内に表示されます。機能の概要と作例は「Luaスクリプトで動かす（Scripting）」を参照してください。

Mug Typography Scriptingプリセット

Character SettingsとPart Settingsを省略し、最大128文字・512パーツをLuaから制御する用途に特化したプリセットです。Global設定と3DのCamera設定は利用できますが、Character / Part選択モード、Copy / Paste、文字・パーツを対象とするRandomizeは表示されません。

インスペクタでの表示 ​

通常プリセットではPageをScriptingへ切り替えて設定します。専用のMug Typography Scriptingプリセットでは、同じScriptingグループがGlobalページ内に常時表示されます。

パラメーター ​

パラメーター説明デフォルト

Paste ScriptOSのクリップボードにあるテキストでScript Textを置き換えます。ホストのUndoで元に戻せます。—

スクリプトの開発はVSCode拡張で

Script Text欄はスクリプトの「供給口」であり、エディタとしての支援機能はありません。開発・デバッグには配布ZIP同梱のVSCode拡張「Mug Typography Scripting Simulator」(MT Sim) を使い、MT SimのCopy ScriptとプラグインのPaste Scriptで完成したスクリプトを移す運用を推奨します。

スクリプトから制御できるもの ​

コールバック書き込み先
OnPreLayoutctx.global（テキスト・フォントサイズ・字送り・行間・整列・縦書き・位置・回転・スケール・塗り・グラデーション・ストローク・影など）、ctx.camera（3D / Scriptingプリセット）
OnLayoutctx.chars[index] / ctx.parts[index]（位置・回転・スケール・ピボット・個別色・個別ストローク・個別影・3D姿勢）、ctx.bounding_box、ctx.output（Write On）

読み取り専用の情報として ctx.time（秒）、ctx.frame、ctx.timeline（クリップ長・進捗）、ctx.font、ctx.meta（APIレベル・プリセット能力・上限値）、文字・パーツの geometry（実測ジオメトリ）が利用できます。完全な仕様は「Luaスクリプティング APIリファレンス」を参照してください。

Edit Pageでの使用上の注意 ​

Generatorプリセットでは、クリップの分割や0フレームより前への拡張に制限があります。

これらの操作を行うと、DaVinci Resolveから正しい時間情報が渡されず、ctx.timeを使うアニメーションが崩れる場合があります。

操作を避けるか、Adjustment Clip（調整クリップ）へFilters側のMug Typographyプリセットを適用してください。

エラー時の挙動と実行制限 ​

スクリプトのエラーは描画を止めません。コールバック内のどこかでエラーが起きると、そのコールバックによる変更全体が破棄され、元の値で描画が続行されます（コールバック単位の原子性）。

失敗箇所結果
スクリプトロードスクリプト全体を実行しない
OnInitializestorageを破棄
OnPreLayoutインスペクタ由来の値で描画し、OnLayout もスキップ
OnLayoutBaseから生成された未変更の値で描画

暴走防止のため、1コールバック呼び出しあたり 5,000,000命令、メモリ 128MiB の上限があります。メモリは必要な分だけ確保されます。上限を超えたコールバックはエラーとして扱われ、上記のとおりロールバックされます。

制限事項

  • スクリプトが書き込んだ値はそのフレームの描画にのみ使われ、インスペクタやキーフレームへは書き戻されません
  • モーフアニメーションの実行中（To Textによる変形中）はスクリプトのコールバックはスキップされます
  • パスジオメトリ自体の書き換えや、外部 .lua ファイルの読み込みには対応していません
