環境設定のカスタマイズ (preferences.json) ​

Mug Typography では、プラグインの最大動作制限（最大文字数・最大パーツ数）を、環境設定ファイル preferences.json を編集することでカスタマイズ可能です。主に、次の2つの目的で調整します。

  • 文字数・パーツ数を増やす: 既存のプリセット（Low / Mid / High）の上限では文字数やパーツ数が足りない場合、maxCharacters / maxParts の値を引き上げることで対応できます。
  • ロード時間を短縮する: 逆に、入力するテキストが短い・パーツ数が少ないことが分かっている場合は、上限値を小さくすることでロード時間を短縮できます。ロード時間は、おおよそ「最大文字数 × 文字パラメーター数 ＋ 最大パーツ数 × パーツパラメーター数」の合計値に比例して長くなるため、必要以上に大きな上限値を設定しないことが、軽快に動作させるコツです。

ファイルの配置場所 ​

環境設定ファイルは、各OSの以下のアプリデータディレクトリに配置されています。ファイルが存在しない場合は、一度プラグインを起動すると自動的にデフォルト設定でファイルが生成されます。

text
%APPDATA%\MugLab\MugTypography\preferences.json
text
~/Library/Application Support/MugLab/MugTypography/preferences.json

※ Windows の場合は、エクスプローラーのアドレスバーに %APPDATA%\MugLab\MugTypography\ と入力してEnterを押すと素早く移動できます。

設定ファイルの形式 ​

ファイルは以下のような JSON 形式です。各プリセット（Low, Mid, High）に対する最大文字数 (maxCharacters) と最大パーツ数 (maxParts) を任意の値にカスタマイズできます。

json
{
  "low": {
    "maxCharacters": 16,
    "maxParts": 60
  },
  "mid": {
    "maxCharacters": 30,
    "maxParts": 84
  },
  "high": {
    "maxCharacters": 60,
    "maxParts": 170
  }
}

CAUTION

環境設定ファイル（preferences.json）の編集に伴うリスクについて

preferences.json による制限値の変更は、DaVinci Resolve で開くすべてのプロジェクト（既存プロジェクトも含む）にグローバルに影響します。そのため、編集する際は以下のリスクを十分に理解した上で行ってください。

  • 値を下げる場合（データ喪失リスク）:
    既存プロジェクトで使用されている文字数やパーツ数よりも「小さい値」に制限値を下げた状態でそのプロジェクトを開いてしまうと、制限値を超えた分の文字やパーツの個別パラメータデータが完全に失われます。 一度保存してしまうと復元できなくなるため、制限値を下げる際は細心の注意を払ってください。
  • 値を上げる場合（パフォーマンスリスク / 動作不安定化）:
    文字数やパーツ数の制限値を引き上げると、より多くの文字を扱えるようになりますが、レンダリング負荷やUI描画負荷が増大します。特に最大の上限値に近い値を設定すると、マシンスペックによっては動作が著しくカクついたり、メモリ不足で DaVinci Resolve がクラッシュしたりする恐れがあります。

制限値に関する仕様

  • 設定可能な最大の上限値:
    maxCharacters（最大文字数）は最大 128 まで、maxParts（最大パーツ数）は最大 512 まで設定可能です。
  • 範囲外の値が設定された場合:
    最大の上限値を超える値や 1 未満の値を記述した場合、その設定項目は無視され、自動的に各プリセットのデフォルト値（Mid の場合は 30文字 / 84パーツなど）に戻ります。
  • Lite版の最大文字数について:
    Lite版（LiteLow / LiteMid / LiteHigh）の最大文字数は、preferences.json で設定した通常版の maxCharacters の値をベースに、自動的に 1.5倍（最大128文字まで）に増やした値が適用されます。
  • 3Dプリセットについて:
    3DはMidのmaxCharacters / maxParts、Lite 3DはMidのmaxCharactersを1.5倍した値を使用します。3D専用の上限項目はなく、Midの設定変更が3D系にも反映されます。

主要キーコード一覧（10進数） ​

設定ファイルに記述する主なキーコードは以下のとおりです。

  • アルファベット（大文字）:
    • A〜Z: 65〜90
    • ※アルファベットを指定した場合、大文字・小文字のどちらのキー入力でも反応します。
  • 数字:
    • 0〜9: 48〜57
  • ファンクションキー:
    • F1〜F12: 65470〜65481
    • F13〜F24: 65482〜65493 （通常のキーボードにない特殊キーに割り当てる際におすすめです）
  • その他特殊キー:
    • Space (スペース): 32
    • Enter (エンター): 65293
    • Escape (エスケープ): 65307
    • Backspace (バックスペース): 65288
    • Delete (デリート): 65535
    • Tab (タブ): 65289
