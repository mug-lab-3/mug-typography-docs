PARAMETER REFERENCE

Copy / Paste

文字・パーツの一部パラメーター、またはエフェクトインスタンス全体を複製し、繰り返しのデザイン作業を短縮します。

⧉Selection Copy選択対象の一部設定を複製↘☑Paste Targets貼り付ける項目を選択↘{}Entire Instance全設定をJSONで複製↘◎Overlayハンドルと輪郭の外観↘

選択中の文字・パーツが持つパラメーター（位置・スケール・色など）を別の文字・パーツへコピー＆ペーストするページです。1か所で調整した見た目を、他の文字へすばやく横展開したいときに使います。

基本的な使い方 ​

Copy / Paste Selection
  1. プレビュー画面で、コピー元となる文字またはパーツを 1 つ選択します。
  2. Copy Selection ボタン（または Alt+Shift+Y）を押します。Copied Source 欄に、現在コピー元として保持している対象が表示されます。
  3. このページの下部にあるチェックボックスで、貼り付けたいパラメーターだけを ON にします。
  4. プレビュー画面で、貼り付け先となる文字・パーツを選択します（複数選択可）。
  5. Paste to Selection ボタン（または Alt+Shift+U）を押すと、チェックした項目だけが貼り付け先に適用されます。

NOTE

コピー元・貼り付け先は、文字どうし／パーツどうしだけでなく、文字とパーツを混在させて貼り付けることもできます。その際、一方にしか存在しないパラメーター（例: 文字専用の Margin After をパーツへ貼り付け）は自動的にスキップされます。

TIP

コピーされるのは、Copy Selection を押したフレーム時点の値です。そのため、同じ文字・パーツを選んだまま別のフレームでペーストすれば、自分自身の「別の時間の設定」をコピー＆ペーストできます。たとえば、あるフレームの状態を後のフレームへ複製してアニメーションを止める、といった使い方が可能です。

パラメーター説明デフォルト
Copy Selection (Alt+Shift+Y)現在選択中の文字・パーツのパラメーター値を、そのフレーム時点の値のままコピー元として記憶します。—
Paste to Selection (Alt+Shift+U)コピー元の、チェックされたパラメーターを現在の選択へ書き込みます。書き込み先は、ペーストした時点のフレームです。—
Copied Source現在コピー元として保持している対象と、コピーしたフレーム番号（例: Char 3 (12f)）を表示する欄です（表示専用・編集不可）。(none)

貼り付け対象の選択（チェックボックス） ​

貼り付ける項目は、以下のチェックボックスで個別に ON/OFF します。(char only) と付いた項目は文字専用で、パーツへの貼り付け時には無視されます。

区分項目
変形（既定 OFF）Offset X / Offset Y / Pivot X / Pivot Y / Scale / Rotation / Stretch X / Stretch Y
カラー（既定 ON）Use Individual Color / Color
文字専用（既定 ON）Margin After / Use Stroke / Stroke Width / Stroke Gap / Stroke Color / Use Shadow Color / Shadow Color

NOTE

Offset・Pivot・Stretch は X と Y を個別にチェックできるため、例えば「X 方向の位置だけを揃える」といった貼り付けも可能です。

NOTE

3Dプリセット固有のZ Offset / Yaw / Pitchは、選択範囲のコピー対象には含まれません。これらを含めて複製する場合は、下記のインスタンス全体コピーを使用してください。

IMPORTANT

ショートカット Alt+Shift+Y / Alt+Shift+U のキー部分は preferences.json でカスタマイズ可能です（Alt+Shift などの修飾キーは変更できません）。

インスタンス全体のコピー ​

**Copy / Paste Entire Instance Settings (Global)**グループでは、現在のエフェクトインスタンスが持つ全パラメーターをJSONとしてOSのクリップボードへコピーし、別のインスタンスへ貼り付けられます。

Entire Instance Settings
パラメーター説明
Copy Entire Settings (Clipboard)現在フレームの設定を含むインスタンス全体のパラメーターをJSONテキストとしてコピーします。
Paste Entire Settings (Clipboard)クリップボード上のJSONを現在のインスタンスへ適用します。

TIP

Output Separation用に同じ設定のノードを複数作る場合や、3D設定を含むデザインを別クリップへ移す場合に便利です。貼り付け後はテキスト、プリセット上限、出力設定を確認してください。
