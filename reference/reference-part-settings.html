PARAMETER REFERENCE

Part Settings

文字を構成する点や画を、パーツ単位で変形・着色します。通常版プリセットで利用できる、最も細かな編集階層です。

↗Transform位置・ピボット・回転↘Z3D Depthパーツごとの奥行き↘◐Colorパーツごとの塗り色↘

パーツ選択モード（Part）で選択したパーツに対して個別の変形を設定するページです。

IMPORTANT

Lite版プリセットにおける制限について
軽量版である Lite プリセット（Lite Low / Lite Mid / Lite High / Lite 3D）では、個別パーツ設定機能が完全に無効化されます。インスペクタ上の本「Part Settings」ページは非表示になり、ビューア上でパーツ単位の選択（Part選択モード）や編集を行うこともできません。パーツごとの個別調整が必要な場合は、通常版プリセット（Low / Mid / High / 3D）をご使用ください。 パラメーターのスロット数は、選択しているプリセット（Low / Mid / High）や preferences.json の設定によって異なります（詳細は付録の「制限事項」を参照）。

グローバル変形 → キャラクター変形 → パーツ変形 の順に重ねて適用されます。

3Dプリセットでの変形 ​

通常の3Dプリセットでは Stretch の代わりに Z Offset が表示され、パーツを奥行き方向へ移動できます。Position、Pivot、Scale、Rotationは引き続き使用できます。

パラメーター説明デフォルト
Z Offsetパーツを文字のZ Offsetからさらに前後へ移動します。キャンバス高さを基準とした値です。0.0

Lite 3DにはPart Settings自体がありません。
