Page: Part Settings ​

パーツ選択モード（Part）で選択したパーツに対して個別の変形を設定するページです。

IMPORTANT

Lite版プリセットにおける制限について
軽量版である Lite プリセット（LiteLow / LiteMid / LiteHigh）では、個別パーツ設定機能が完全に無効化されます。インスペクタ上の本「Part Settings」ページは非表示になり、ビューア上でパーツ単位の選択（Part選択モード）や編集を行うこともできません。パーツごとの個別調整が必要な場合は、通常版プリセット（Low / Mid / High）をご使用ください。 パラメーターのスロット数は、選択しているプリセット（Low / Mid / High）や preferences.json の設定によって異なります（詳細は付録の「制限事項」を参照）。

グローバル変形 → キャラクター変形 → パーツ変形 の順に重ねて適用されます。

変形パラメーター ​

パラメーター説明
Posパーツ個別の位置オフセット（X, Y）。
Pivotパーツのスケール・回転の基準点（X, Y）。
Scaleパーツを均等にスケールします。
Rotationパーツの回転角度。
Stretchパーツを X / Y 方向に個別スケールします。

個別の色 ​

パラメーター説明
Use Indiv ColorON にすると、グローバル・文字の色設定を上書きしてこのパーツ専用の色を使います。
Colorパーツごとの色（RGB + 透明度）。Use Indiv Color が ON のときのみ有効。
