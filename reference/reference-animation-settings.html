PARAMETER REFERENCE

Animation Settings

キーフレームの作成・整理から、輪郭を動かすデフォーム、手描き風の揺れ、ランダムな初期状態までをまとめます。

◆Keyframes打刻・整理・範囲削除↘≈Deformation4種類のパス変形↘✎Hand Drawn輪郭と姿勢の揺らぎ↘※Randomize選択対象を範囲内で変化↘

キーフレームの設定・削除・最適化・ランダマイズと、手描き風エフェクトおよびパス変形エフェクトを設定するページです。

CAUTION

Edit ページで分割（カット）したクリップへのキーフレーム設定について
Edit ページで分割したクリップに対して、本プラグインからキーフレームを設定（オートキーフレーム、ビューア上の操作、WinAlt+K/MacOption+K による打刻など）すると、キーフレームが正しい位置に設定されません。これは DaVinci Resolve 側の仕様による問題で、プラグイン側では対応できません。このため、本プラグインを適用するクリップは分割しない運用を推奨します（詳細は付録の「制限事項」を参照）。

オートキーフレーム ​

パラメーター説明
Auto Key FrameON にすると、パラメーターを変更するたびに自動的にキーフレームが設定されます。

NOTE

移動・回転などの操作のたびにキーフレームが記録されるため、意図せず大量のキーフレームが作成されたり、意図しないアニメーションが設定されることがあります。アニメーションを付けない場合は OFF のまま使用してください。

キーフレームの手動設定 ​

パラメーター説明
Set Keyframes for Selected (Alt+K)現在選択中の文字・パーツに対して、現在フレームの値をキーフレームとして記録します。値は変更しません。（ショートカットキー WinAlt+K/MacOption+K でも実行可能）

NOTE

記録される範囲は選択モードによって異なります。いずれも、その範囲内のアニメーション可能なすべてのパラメーターにキーフレームが設定されます。

  • Global: エフェクト全体（Global Settings に加え、全文字・全パーツ・カメラ・各エフェクトなどを含むすべて）
  • Character: 選択中の文字の Character Settings 全項目（Pos / Pivot / Scale / Rotation / Stretch（3Dプリセットでは Z Offset / Yaw / Pitch）/ Margin After / Color / Stroke Width / Stroke Gap / Stroke Color / Shadow Color）
  • Part: 選択中のパーツの Part Settings 全項目（Pos / Pivot / Scale / Rotation / Stretch（3Dプリセットでは Z Offset）/ Color）

Use Indiv Color などの ON/OFF スイッチはアニメーション非対応のため記録されません。

冗長キーフレームの整理（Clean Up Redundant Keyframes） ​

すべてのアニメーションパラメータから、値が変化しない不要なキーフレームを自動で検出して一括削除し、整理する機能です。

パラメーター説明
Clean Up Redundant Keyframesクリックすると確認ダイアログが表示され、全パラメータをスキャンして冗長な（すべてのキーで同じ値を持つ）キーフレームを一括削除し、パラメータを単一値（キーフレームなし）に整理します。

NOTE

移動や変形時の「オートキーフレーム」などで意図せず大量に記録されてしまったキーフレームを整理するのに便利です。不要なキーフレームを削減することで、動作パフォーマンスの向上が期待できます。 完了後に削除されたキーフレーム数がダイアログで通知されます。

キーフレームの削除（Delete Keys） ​

指定した時間範囲内のキーフレームをまとめて削除できます。

パラメーター説明
Time Mode範囲指定を Relative（相対フレーム）か Absolute（絶対フレーム）で行うか選びます。
Start (frame)削除範囲の開始フレーム。
Start (sec)開始位置の秒数表示（参照用）。
End (frame)削除範囲の終了フレーム。
End (sec)終了位置の秒数表示（参照用）。
Delete Keys in Range (Selected)選択項目の指定範囲内キーフレームをすべて削除します。

パス変形エフェクト（Path Deformation Effects） ​

手描き風エフェクトに加えて、文字のアウトライン（輪郭）を規則的に歪ませる、4つの変形エフェクトが用意されています。 それぞれを ON（Enabled）にすることで、パラメータに応じた独自の変形・アニメーション効果を適用できます。

1. パルスエフェクト（Pulse Effect） ​

文字の輪郭を外側・内側へ伸縮させ、呼吸しているかのように太くなったり細くなったりする効果です。

2. 台形変形エフェクト（Trapezoid Effect） ​

文字を任意のアングル（角度）の軸に沿って台形に歪ませる（遠近感を出す）効果です。 軸を傾けることで、すぼまる方向を自由に変更でき、時間経過で台形と逆台形を往復するような動きを作成できます。

3. 球面化エフェクト（Sphere Effect） ​

文字のアウトラインを球面上に投影し、バルーンのように中央をプクッと膨らませたり、逆にピンチ（すぼめる）したりする効果です。

4. 樽型歪みエフェクト（Barrel Effect） ​

文字の中心から外側に向けてスケールを徐々に変化させ、樽のように中央が膨らんだ歪み（または糸巻き型の歪み）を与える効果です。

手描き風エフェクト（Hand Drawn Effect） ​

テキストに有機的な手描き風のアニメーションを付加する機能です。文字の輪郭線（パス）をランダムに波打たせる「ウォブル」と、文字の位置・回転・スケールをフレームごとに微妙にずらす「ジッター」の2種類の効果を組み合わせて、手書きで書いたようなゆらぎを表現します。

パラメーター説明デフォルト
EnabledON にするとエフェクトを有効にします。アニメーション可能。OFF
Pattern Countループするパターンのコマ数。パラパラ漫画のコマ数のイメージで、この数のパターンを順に切り替えながら繰り返します。3
Wobble Amplitude輪郭線の波打ち（ウォブル）の強さ。0 でウォブルなし。アニメーション可能。3.0
Fine Wobble手書きの微細な手の震え（局所的なブレ）の強さ。0 で震えなし。アニメーション可能。0.15
Jitter Amplitude文字の位置ずれ（ジッター）の強さ。0 でジッターなし。アニメーション可能。2.0
Frame Rateエフェクトのパターンが切り替わる速度（回/秒）。低い値ほど変化がゆっくりになります。アニメーション可能。6.0
Seedウォブル・ジッターのランダムパターンを決定するシード値。値を変えると別のパターンになります。0
Targetsエフェクトを適用する対象の文字・パーツを指定します。記述方法は「Targets（対象指定）の記述フォーマット」を参照してください（文字単位 c1 / パーツ単位 p5 / 範囲 c1-c3 / 否定 !c2 などが使用できます）。空欄の場合はすべてのパーツに適用されます。(空)

TIP

使いこなしのヒント

  • まず Enabled を ON にすれば、デフォルト値のまま自然な手描き風のゆらぎが付きます。
  • 揺れが強すぎると感じたら、Wobble Amplitude（輪郭の波打ち）と Jitter Amplitude（位置ずれ）を下げてください。
  • 揺れの速さは Frame Rate で調整します。低くするとゆっくりした動きになります。
  • Pattern Count を 1 にすると、ゆらぎが静止した（動かない）手描き風の見た目になります。動かさず質感だけ欲しいときに便利です。

ギザギザ・トゲトゲにする例

ランダマイズ（Randomize） ​

選択した文字・パーツに対して、指定した範囲内のランダムな値を一括で適用し、同時にキーフレームとして記録します。アニメーションの初期状態を素早く作るのに便利です。

パラメーター説明デフォルト範囲
Pos XX 位置をランダム化（ON/OFF + Min / Max）0.4 ～ 0.6
Pos YY 位置をランダム化（ON/OFF + Min / Max）0.4 ～ 0.6
Scaleスケールをランダム化（ON/OFF + Min / Max）0.7 ～ 1.3
Rotation回転をランダム化（ON/OFF + Min / Max）-30° ～ +30°
Stretch XX 方向ストレッチをランダム化（ON/OFF + Min / Max）0.8 ～ 1.2
Stretch YY 方向ストレッチをランダム化（ON/OFF + Min / Max）0.8 ～ 1.2
Randomize Selected (Set Key)チェック済み項目を選択文字・パーツに適用し、キーフレームとして記録します。

NOTE

Min / Max に指定する値は絶対値です。現在の値へのオフセットではなく、指定した範囲内の値がそのままパラメーターにセットされます。

ランダマイズの手順：

  1. ランダム化したい項目の ON/OFF を有効にする
  2. Min / Max 値を設定する
  3. プレビュー画面でランダム化したい文字・パーツを選択する
  4. Randomize Selected (Set Key) ボタンを押す
