LAYOUT & INTERACTION

Move Even Character Parts Exactly as You Want

Break down text into individual characters and even into the parts that make up each font glyph. Directly design intricate effects in the viewer, such as tilting only part of a title or making only a dakuten mark pop out.

  • Direct manipulation in the viewer
  • Character and part level
  • Marquee multiple selection
How to UseSettings Reference

How to Use ​

  1. Set Selection Mode at the top of the Inspector to match what you want to move: Global for the entire text, Character for individual characters, or Part for parts within a character.
  2. Click the target you want to move in the viewer. To select multiple targets, drag a marquee from an empty area, or WinCtrl+Alt/MacCommand+Option + click to add or remove targets one at a time.
  3. Drag the target to move it.
  4. WinCtrl/MacCommand + click a selected target, then use the handles to adjust its rotation, scale, and pivot.
  5. To fine-tune the values numerically, switch Page to the settings page that matches the selection mode (Global Settings / Character Settings / Part Settings).

TIP

It is efficient to arrange elements roughly in the viewer, then refine their values in the Inspector. Viewer controls and Inspector values edit the same parameters.

Choosing Between Characters and Parts ​

What You Want to DoSelection ModeEditing Page
Adjust the font or the entire textGlobalGlobal Settings
Adjust only specific charactersCharacterCharacter Settings
Adjust dots or strokes within a characterPartPart Settings

Part is available only in standard-edition presets. Part Settings and the Part selection mode are not displayed in the Lite edition. If you need to edit parts, use the standard edition from the outset.

TIP

If you started in the Lite edition and later need part-level editing, you don't have to rebuild your work. Add a new standard-edition instance, then use Copy / Paste Entire Instance Settings on the Copy / Paste page to copy your Lite-edition settings as JSON and paste them in, switching presets without losing your design. Note that this copy only carries the values at the current frame — keyframed animation is not transferred, so you'll need to rebuild any animation after switching.

NOTE

Characters and parts excluded from rendering in Output Settings cannot be selected in the viewer either.
