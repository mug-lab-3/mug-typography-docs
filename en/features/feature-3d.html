3D TYPOGRAPHY NEW

Pseudo-3D that brings out depth

Tilt the entire text as a plane, or place individual characters and parts at different depths. Combine them with the dedicated camera and perspective controls to create dimensional typography while retaining the feel of a 2D workflow.

  • Camera controls in the viewer
  • Character and part depth
  • Works with existing effects
How to UseSettings Reference

How to Use ​

  1. Place Mug Typography 3D or Mug Typography Lite (3D) from the Effects Library.
  2. Set Selection Mode to Camera.
  3. Orbit the camera using the Orbit handle in the viewer, or WinAlt/MacOption + drag an empty area.
  4. Adjust perspective with Perspective Strength and on-screen size with Camera Zoom.
  5. Refine the composition with the Target Pad or Lens Shift as needed.

TIP

Use Lens Shift to adjust the on-screen composition Moving the Position in each level (Global/Character/Part) changes the relationship with the camera, which also changes the perspective distortion. If you want to adjust only the position on the screen while keeping the three-dimensional appearance, using Lens Shift in the camera settings makes it easier to organize the composition.

3D and Lite 3D Presets ​

PresetMaximum CharactersPart SettingsBest For
Mug Typography 3D30Up to 84 partsDimensional compositions with depth for individual parts
Mug Typography Lite (3D)45NoneCharacter-level 3D and longer text

Both use limits based on the standard Mid preset. Lite 3D omits Part Settings and can render up to 512 parts, but does not let you edit Z Offset for individual parts.

TIP

You can change the maximum character and part limits with maxCharacters / maxParts in preferences.json. See Customizing Preferences (preferences.json) in the appendix for instructions.

Creating Depth at Three Levels ​

Level3D-Specific ParametersPurpose
GlobalGlobal Yaw / Global PitchTilts the plane of the entire text
CharacterZ Offset / Yaw / PitchChanges the depth and orientation of each character
PartZ OffsetSeparates dots, strokes, and other parts forward or backward
CameraTarget / Yaw / Pitch / Perspective / Zoom / Lens ShiftDetermines the viewing position and projection of the scene

TIP

A straightforward workflow is to set the overall orientation with Global Yaw/Pitch, create front-to-back variation with character Z Offset, and adjust the camera last.

Differences from the 2D Presets ​

Unavailable Settings ​

In 3D presets, Stretch is not displayed and cannot be used in Character Settings or Part Settings. Stretch remains available in Global Settings.

NOTE

Crayon fill can still be used in 3D presets, but the scribble pattern changes every time you move the camera or the characters/parts.

Additional 3D Settings ​

SettingsAdded Parameters
Character SettingsZ Offset / Yaw / Pitch
Part SettingsZ Offset

These settings do not replace Stretch. They adjust depth and orientation in 3D. Position, Pivot, Scale, Rotation, color, stroke, and other settings remain available in 3D presets.

Gradients, shadows, bounding boxes, deformations, morphs, and Write On also follow the appearance after 3D projection.

TIP

On the Copy / Paste page, Z Offset and the character-specific Yaw / Pitch can each be selected for pasting.

Viewer Controls ​

When Selection Mode is Global, Character, or Part, the text is displayed before camera settings are applied. Viewer operations such as moving, rotating, and scaling also act on the 2D coordinates in this state. This lets you edit the layout with the same feel as a 2D preset, unaffected by camera tilt or perspective.

To view and adjust the final composition with the camera applied, switch Selection Mode to Camera.
