OUTPUT & COMPOSITING

Send every character into a different world

Extract only selected characters or parts over a transparent background and connect them to separate effects. Split one typography composition into as many as three streams for complete compositing freedom.

  • Target characters and parts
  • Up to three output streams
  • Composite freely in Fusion
How to UseSettings Reference

How It Works ​

This feature does not add multiple output terminals to one instance. Instead, duplicate instances with the same settings and change each one's Output Mode to create the normal output and separated outputs.

Output ModeRendered Content
DefaultNormal output excluding separated targets
Separation 1 OnlyOnly targets included in Separation 1 Targets
Separation 2 OnlyOnly targets included in Separation 2 Targets

How to Use ​

  1. Set Page to Output Settings.
  2. Turn on Enable Output Separation.
  3. Enter the target to extract in Separation 1 Targets, for example c2.
  4. Duplicate the node as an instance node (its settings and animation stay synchronized automatically).
  5. On the duplicate, right-click Output Mode and choose Deinstance to unlink it, then set one instance to Default and the other to Separation 1 Only.
  6. Add any effects you want to the separated output, then Merge it with the normal output.

Writing Targets ​

Specify characters with c and parts with p. Numbers start at 1.

text
c2           # Character 2
c1-c3        # Characters 1 through 3
p1 p4 p7     # Parts 1, 4, and 7
c1 !p2       # Character 1, excluding part 2

NOTE

A blank Separation Targets field means “no targets.” By contrast, a blank deformation Targets field means “all targets,” so be mindful of the difference between pages.

Splitting into Up to Three Streams ​

Using Separation 1 and Separation 2 together lets you create up to three visual streams including the normal output. Create a duplicate for each Output Mode and composite them at the end.
