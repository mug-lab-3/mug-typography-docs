AUTO ALIGNMENT

Tidy up a messy layout with a single button

Automatically recalculates the layout to match each character's scale and rotation. Free yourself from adjusting positions one character at a time — titles that mix characters of different sizes line up cleanly.

  • Scale-aware placement
  • Horizontal and vertical
  • Shortcut support
How to UseSettings Reference

How to Use ​

  1. Set Selection Mode to Character.
  2. Select multiple characters to align in the viewer.
  3. Set Page to Character Settings.
  4. Press Align Selected Horizontally or Align Selected Vertically at the top of the page.

The shortcuts are WinAlt+Ctrl+H/MacOption+Command+H for horizontal alignment and WinAlt+Ctrl+V/MacOption+Command+V for vertical alignment.

NOTE

The leftmost character (for horizontal alignment) or the topmost character (for vertical alignment) serves as the reference. The reference character does not move; the other characters are repacked against it without gaps.

How Alignment Works ​

Alignment DirectionExample
Align Selected Horizontally
Align Selected Vertically
  • Horizontal alignment aligns each character's baseline (vertical position); vertical alignment aligns the center axis of the column (horizontal position).
  • Both account for each character's scale when calculating spacing, so characters of different sizes neither overlap nor drift apart.

NOTE

Running an alignment resets each character's Pivot to the default (0.5, 0.5).

IMPORTANT

Alignment works only while Character selection mode is active in the viewer. It does not work in Global or Part selection mode.
