MORPH & TRANSITION

One word flows into the next

Morph smoothly from one text to another, with letter shapes matched automatically. Place just two keyframes on Progress to create a transition in which the characters melt into one another.

  • Two keyframes to finish
  • Natural morphing algorithm
  • Works across text lengths
How to UseSettings Reference

How to Use ​

  1. Set Page to Global Settings.
  2. Enter the source string in Text.
  3. Turn on Enable Morph in Morph Animation.
  4. Enter the destination string in To Text.
  5. At the starting frame, set Progress to 0 and add a keyframe.
  6. At the ending frame, set Progress to 1.

0 is the shape of Text, and 1 is the shape of To Text. Intermediate values interpolate the parts between the two. Adjust the spacing and curve between keyframes to control the feel of the speed.

Chaining Multiple Morphs ​

You can also morph through three or more strings in sequence, like A→B→C.

  1. On the frame where the first morph completes, change Text to the same string as To Text and set a keyframe, then reset Progress to 0.
  2. On the same frame, change To Text to the next string and set a keyframe as well, then animate Progress to 1 again.

The shape shown at Progress 1 is identical to the shape right after swapping Text and resetting to 0, so the switch is invisible on screen. Repeat this to chain as many transitions as you like.

Adjusting the Matching ​

When the character counts or shapes differ substantially, the plugin estimates which source parts should transform into which destination parts.

ParameterWhen to Adjust It
Matching StrengthWhen you want matching to prioritize parts with similar filled areas. At 0, positional proximity is prioritized.
Same Glyph PriorityWhen both strings contain the same character and you want those matching characters connected first.

Tips for a Clean Result ​

  • Check the motion with the default values first, and change parameters such as Matching Strength gradually only if parts cross unnaturally.
  • Increase Same Glyph Priority if you want the same character to remain in the same position.
  • For strings with very different shapes, shorten the time spent around the middle of Progress to make the complex transformation less noticeable.
  • If the vertical position wobbles, try Baseline for Vertical Alignment.
  • If matching does not behave as intended, try changing Horizontal Alignment from Center to Left or Right. Changing the positional relationship between characters also changes the matching result.

NOTE

How parts are divided depends on the font's outline structure. Even with the same strings, changing the font can change the morph matching and motion.
