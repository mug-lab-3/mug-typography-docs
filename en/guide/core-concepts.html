Core Concepts ​

This plugin has three levels that you can work with.

LevelDescriptionExample
GlobalSettings applied to the entire text at onceFont, overall position, rotation, and scale
CharacterSettings that can be applied individually to each characterEnlarge a specific character or change its color
PartSettings for each path element that makes up a characterMove only the diacritic marks in a character

For example, you can move the entire text to the right globally, shift only a specific character farther upward, and rotate only a particular part within that character.

How Transformations Accumulate ​

Transformations at each level accumulate in the order Global → Character → Part, and the result is rendered.

MGlobalRotation +10°Around the pivot of the entire text
×
MCharacterRotation +15°Then around the character's pivot
×
MPartRotation +20°And then around the part's pivot
＝
MFinalOrientation +45°The orientation totals 45°, while the rendered position depends on the relative locations of the pivots and is not a simple sum
  • Sun = Global (entire text)
  • Earth = Character (each character)
  • Moon = Part (each part)

The Earth revolves around the Sun, and the Moon revolves around the Earth. Just as the Moon's position and orientation as seen from the Sun are determined by both rotations, Global, Character, and Part transformations are multiplied in sequence around the pivot at each level. Therefore, although the rotation angles can be added together, the final position is not a simple sum.

Transformations accumulating in the order Global → Character → Part

Color composition follows the same order of precedence. For details, see "Color Priority" in the appendix.

When Part Controls Are Not Needed ​

If you do not need part-level controls such as movement or scaling, selecting a Lite version makes clip loading and selection more responsive. This is because Part Settings are omitted from the Inspector, reducing the number of controls.

It is important to choose the preset that suits your purpose. For details, see "Getting Started".

Understanding the Indices ​

Character 1, 2... in Character Settings indicates the character number counted from the beginning of the input text. Part Setting 1, 2... in Part Settings indicates the part number across all characters. These correspond to c1 and p1 used in Targets under Output Settings and Animation Settings.

TIP

Each item in the Inspector displays the actual character and the part's position within that character. If the numbers alone are difficult to interpret, select the target in the viewer and check the corresponding item.
