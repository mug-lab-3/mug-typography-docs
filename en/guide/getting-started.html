Getting Started ​

This plugin provides six presets for 2D and two presets for 3D. First, choose between flat typography or a 3D presentation using a camera and depth, then select a preset based on whether you need part editing and the amount of text.

In the Trial version, the maximum number of characters is uniformly limited to 8, regardless of the selected preset.

STEP 1: Choose 2D or 3D ​

Standard

Mug Typography 2D

For standard titles, subtitles, and motion graphics

Edit characters and parts on a flat plane, primarily using position, rotation, scale, and stretch. You can choose a processing limit from Low / Mid / High.

Mug Typography 3D

For presentations using a camera, perspective, and depth

Includes Camera Settings and 3D transformations. Characters and parts can be positioned along the Z-axis. Its limits are equivalent to Mid. For details, see “Create 3D Typography.”

STEP 2: Choose the standard or Lite version ​

Mug Typography Standard

For detailed control of individual parts

Includes Part Settings (individual part adjustments). You can configure parameters for each part that makes up a character.

Mug Typography Lite

For users who do not need part-level editing

Omits Part Settings for faster loading and processing. The maximum character count is 1.5 times that of the standard version, and up to the rendering limit of 512 parts can be handled.

STEP 3: For 2D, choose a size (Low / Mid / High) ​

The 3D version has fixed limits equivalent to Mid. If you select the 2D version, choose a size based on the length of the text and the complexity of its characters.

Together, the six 2D types and two 3D types provide a total of eight available presets.

How they appear in the DaVinci Resolve Effects Library

TIP

The more parameters a preset has, the slower the response becomes when switching clips or nodes. Choose the preset that fits the number of characters and parts you actually need. Each preset's limits can also be changed via maxCharacters / maxParts in preferences.json (for instructions, see “Customizing Preferences (preferences.json)” in the appendix).

NOTE

If a limit is exceeded, anything beyond that limit will not be rendered.

  • If the maximum character count is exceeded: Characters beyond the limit, counted from the beginning, will not be rendered.
  • If the maximum part count is reached: From the character at which the limit is reached onward, entire characters will not be rendered (a character will never be rendered with only some of its parts missing).

The currently active limits are always displayed in the Version field at the top of the Inspector in the format (C: character count, P: part count). If text is not displayed, check this field first.

Edit Page ​

Under Open FX in the Effects Library, the eight presets above are available in the following two locations to suit different uses. Place or apply them on the timeline as needed.

Generator

  • Effect Library
  • Open FX
  • Generators
  • MugLab
  • Mug Typography (Low / Mid / High / Lite / 3D…) All 8 types

Place as a new clip

Use this when you want to place the plugin directly on the timeline as a new clip. You can drag and drop it onto the timeline.

Filter

  • Effect Library
  • Open FX
  • Filters
  • MugLab
  • Mug Typography (Low / Mid / High / Lite / 3D…) All 8 types

Apply as an effect to an existing clip

Use this when you want to apply the plugin as an effect to an existing clip. On other pages, such as the Color Page, only this version (Filters) is displayed.

Enable controls in the Viewer ​

To use on-screen handles in the preview (for moving, rotating, scaling, and so on), click the boxed area at the bottom left of the timeline → select Open FX Overlay.

Example of use on the Edit Page (select Open FX Overlay using the switch at the bottom left)

NOTE

The Edit Page has limitations imposed by DaVinci Resolve. For details, see “Limitations” in the appendix.

Fusion Page ​

On the Fusion Page, add the node for the required preset to the node graph from the following location. All eight presets available on the Edit Page can also be added as nodes.

  • Tools
  • OpenFX
  • MugLab
  • Mug Typography (Low / Mid / High / Lite / 3D…) All 8 types

The node's Input port does not need to be connected — the node works on its own. When an input is connected, the input image is rendered as the background, with the text composited on top.
