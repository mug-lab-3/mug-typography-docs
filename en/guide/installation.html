Installation ​

After extracting the distributed zip file, you will find the MugTypography.ofx.bundle folder and installer scripts. Simply double-click the appropriate script to complete the installation.

  1. Quit DaVinci Resolve if it is running.

  2. Extract the zip file.

  3. Double-click install-win.bat. A security warning stating that the publisher could not be verified will appear. Click Run.

  4. When the administrator permission confirmation dialog appears, select Yes.

  5. When the installation completion message appears, launch DaVinci Resolve.

  1. Quit DaVinci Resolve if it is running.

  2. Extract the zip file.

  3. Double-click install-mac.command. macOS Gatekeeper will display a dialog saying it was not opened. Click Done to close it.

  4. Open System Settings → Privacy & Security. Near the bottom, you will see a message that "install-mac.command" was blocked and an Open Anyway button. Click it.

  5. A confirmation dialog asking whether you want to open it will appear. Click Open Anyway.

  6. A password dialog will appear. Enter your Mac login password and click OK. The dialog says that osascript wants to make changes. osascript is a standard macOS mechanism used by this installer to copy the plugin, so this is expected.

  7. When the installation completion message appears, launch DaVinci Resolve.

Verifying the Installation ​

From the DaVinci Resolve menu, open DaVinci Resolve → Preferences → Video Plugins. Installation is complete if the status of MugTypography.ofx.bundle is Loaded successfully.
