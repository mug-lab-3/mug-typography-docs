PARAMETER REFERENCE

Character Settings

Apply individual position, rotation, color, stroke, and shadow settings to selected characters. This page covers everything from aligning multiple characters to styling individual characters.

≡AlignAlign multiple characters horizontally or vertically↘↗TransformPosition, pivot, and rotation↘◐ColorPer-character fill color↘◇StrokePer-character stroke↘◒ShadowPer-character shadow color↘

This page configures individual transforms for characters selected in Character selection mode. The number of parameter slots varies depending on the selected preset (Low / Mid / High) and the settings in preferences.json (see Limitations in the appendix for details).

Character transforms are applied in addition to the global transform.

Aligning Characters (Align Selected) ​

These buttons neatly rearrange multiple characters selected in the viewer while accounting for character scale. They are located at the top of the Inspector page.

IMPORTANT

Selection mode limitation
Character alignment works only when Character selection mode is active in the viewer. It does not work in Global or Part selection mode.

ParameterDescriptionDefault Shortcut
Align Selected HorizontallyUses the left edge of the leftmost selected character as the reference and places the other characters to its right without gaps. The characters are aligned to the same baseline (vertical position), and each character's Pivot is reset to the default (0.5, 0.5).WinAlt+Ctrl+H/MacOption+Command+H
Align Selected VerticallyUses the top edge of the topmost selected character as the reference and places the other characters below it without gaps. The center axes of the column (horizontal positions) are aligned, and each character's Pivot is reset to the default (0.5, 0.5).WinAlt+Ctrl+V/MacOption+Command+V

NOTE

Why the shortcuts use Alt + Ctrl
Unlike standard shortcuts such as Alt plus a letter, these shortcuts use Alt + Ctrl. DaVinci Resolve gives priority to host functions that consume combinations such as Alt + V for Paste Attributes. The extra modifier avoids that conflict and lets the plugin reliably capture the keystroke.
The assigned shortcut keys can be customized in the preferences.json configuration file.

Individual Character Settings (Character 1, 2...) ​

Expand a group such as “Character 1” to display individual settings for the selected character.

Reset ​

ParameterDescription
Reset This CharacterResets the transforms, margin, individual color, individual stroke, and individual shadow color of the character and all its parts to their default values. For 3D presets, this also resets the character's Z Offset / Yaw / Pitch and each part's Z Offset.

Margin After ​

ParameterDescription
Margin After (em)Specifies in em units how much space to add after the character.

Transform Parameters ​

ParameterDescription
PosPer-character position offset (X, Y).
PivotReference point for character scaling and rotation (X, Y).
ScaleScales the character uniformly.
RotationCharacter rotation angle.

Stretch in 2D Presets ​

ParameterDescriptionDefault
StretchScales the character independently along the X and Y axes. It is not displayed for 3D presets.X1.0Y1.0

Transforms in 3D Presets NEW ​

In 3D presets, Stretch is not displayed and cannot be used.

The following parameters are added to adjust character depth and tilt in 3D.

ParameterDescriptionDefault
Z OffsetMoves the character along the depth axis. The value is relative to the canvas height.0.0
YawTilts the character left or right around its Pivot. With a positive value, the right edge moves away from the camera.0°
PitchTilts the character up or down around its Pivot. With a positive value, the top edge moves away from the camera.0°

Position, Pivot, Scale, and Rotation are also available in 3D presets. See Camera Settings for overall camera settings.

Individual Color ​

ParameterDescription
Use Indiv ColorWhen ON, overrides the global color setting and uses a color specific to this character.
ColorPer-character color (RGB + opacity). Available only when Use Indiv Color is ON.

Individual Stroke ​

ParameterDescriptionDefault
Use Indiv StrokeWhen ON, overrides the global stroke settings and uses stroke settings specific to this character. Turning it ON displays the parameters below.OFF
Stroke WidthSpecifies the character-specific stroke width as a ratio of the font size.0.0
Stroke GapSpecifies the gap between the character-specific fill and stroke as a ratio of the font size.0.0
Stroke ColorSpecifies the character-specific stroke color.Off Black#1c1c1e

Individual Shadow Color ​

ParameterDescriptionDefault
Use Indiv Shadow ColorWhen ON, overrides the global shadow color setting and uses a shadow color specific to this character. Turning it ON displays the parameter below.OFF
Shadow ColorSpecifies the character-specific shadow color (RGB + opacity). Available only when Use Indiv Shadow Color is ON.Blue#5555ff

NOTE

A character with its individual shadow color enabled casts a shadow even when Draw Global Shadow in Global Settings is OFF. To add a shadow only to specific characters, leave the global shadow OFF and turn ON Use Indiv Shadow Color for the target characters. The Distance and Angle values from Global Settings are shared by all shadows. Only the color can be changed individually.
