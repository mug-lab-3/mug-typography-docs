Lite tiers: Automatically relaxed to 1.5x of the normal presets (LiteLow: 24, LiteMid: 45, LiteHigh: 90).

Edit Page: Keyframe EditorOnly parameters of the currently active page in the Inspector are shown. To set keyframes for parameters on other pages, switch pages accordingly.

If you need to customize animation curves or easing for movement, please use the Fusion page version of the plugin.

This is an issue caused by DaVinci Resolve's behavior and the plugin cannot work around it. For this reason, we recommend not splitting the clip you apply this plugin to.
TTC Font List ChangeStarting with v1.5, all faces contained in .ttc files are now listed individually in the font list. As a result, projects created with older versions may have shifted font selections if any font listed after a .ttc file was used. If this occurs, please re-select the font in the Inspector.

Customizing Preferences (preferences.json) ​

In Mug Typography, you can customize the plugin's maximum operating limits (maximum characters and maximum parts) by editing the configuration file preferences.json. There are two main reasons to adjust these values:

  • Increasing the character/part limits: If the limits of the existing presets (Low / Mid / High) aren't enough for your text or part count, you can raise the maxCharacters / maxParts values to accommodate it.
  • Reducing load time: Conversely, if you know your text will be short or use few parts, you can lower the limits to reduce loading time. Load time is roughly proportional to "(max characters × number of character parameters) + (max parts × number of part parameters)", so the key to keeping the plugin responsive is not to set the limits higher than you actually need.

File Location ​

The preference file is located in the application data directory of each OS. If the file does not exist, it will be automatically created with default settings upon launching the plugin for the first time.

  • Windows: %APPDATA%\MugLab\MugTypography\preferences.json
    (You can open this folder by entering %APPDATA%\MugLab\MugTypography\ in the Address Bar of Explorer and pressing Enter.)
  • macOS: ~/Library/Application Support/MugLab/MugTypography/preferences.json

Configuration Format ​

The file is written in JSON format. You can customize the maximum characters (maxCharacters) and maximum parts (maxParts) for each preset (Low, Mid, High) by changing their values:

json
{
  "low": {
    "maxCharacters": 16,
    "maxParts": 60
  },
  "mid": {
    "maxCharacters": 30,
    "maxParts": 84
  },
  "high": {
    "maxCharacters": 60,
    "maxParts": 170
  }
}

CAUTION

Risks Associated with Editing Preferences (preferences.json)

Changes made to preferences.json globally affect all projects (including existing ones) in DaVinci Resolve. Please understand the following risks before making any modifications:

  • Lowering the Limit (Data Loss Risk):
    If you lower the maximum limits below the number of characters or parts already used in an existing project, and then open that project, any character or part parameter data exceeding the new lower limit will be permanently lost (truncated). Saving the project in this state will overwrite the data irreversibly. Use extreme caution when lowering limits.
  • Raising the Limit (Performance & Stability Risk):
    Increasing the limits allows you to handle longer text, but significantly increases rendering and UI overhead. Setting limits close to the hard maximums may cause sluggish performance or lead to DaVinci Resolve crashes due to memory exhaustion, depending on your system specifications.

Specifications

  • Maximum Limits:
    maxCharacters (max characters) is up to 128, and maxParts (max parts) is up to 512.
  • Out-of-range Values:
    If a value exceeding these limits or a value less than 1 is specified, it will be ignored and automatically return to the default value for the respective preset (e.g., 30 characters / 84 parts for Mid).
  • Max Characters for Lite Presets:
    The maximum character limit for Lite presets (LiteLow / LiteMid / LiteHigh) is automatically increased to 1.5 times (up to 128 characters) the value of maxCharacters configured for the normal presets.

Common Keycodes (Decimal) ​

Common keycodes for use in the configuration file are listed below.

  • Alphabets (Uppercase):
    • A to Z: 65 to 90
    • Note: When a letter is specified, it responds to either uppercase or lowercase key input.
  • Numbers:
    • 0 to 9: 48 to 57
  • Function Keys:
    • F1 to F12: 65470 to 65481
    • F13 to F24: 65482 to 65493 (Recommended for mapping to macro keys on custom input devices)
  • Other Special Keys:
    • Space: 32
    • Enter: 65293
    • Escape: 65307
    • Backspace: 65288
    • Delete: 65535
    • Tab: 65289

Transform Order ​

Global Transform (Position / Rotation / Scale / Stretch)
    ↓ (applied additionally)
Character Transform (Pos / Rotation / Scale / Stretch)
    ↓ (applied additionally)

In 3D presets, Global Yaw/Pitch, Character Z Offset/Yaw/Pitch, and Part Z Offset are added to the 2D transforms at their respective levels, followed by the Camera Settings projection.
