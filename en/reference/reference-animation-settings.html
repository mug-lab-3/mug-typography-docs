PARAMETER REFERENCE

Animation Settings

Create and clean up keyframes, deform outlines, add hand-drawn motion, and randomize selected elements.

◆KeyframesSet, clean up, and delete ranges↘≈DeformationFour path deformation effects↘✎Hand DrawnOutline and pose variation↘※RandomizeVary selected elements within a range↘

Configures, deletes, and optimizes keyframes, applies randomization, and controls the hand-drawn animation and path deformation effects.

CAUTION

About setting keyframes on clips that have been split (cut) on the Edit page
If you set keyframes from this plugin (auto keyframe, viewer operations, WinAlt+K/MacOption+K, etc.) on a clip that has been split on the Edit page, the keyframes are not placed at the correct positions. This is an issue caused by DaVinci Resolve's behavior, and the plugin cannot work around it. For this reason, we recommend not splitting the clip you apply this plugin to (see "Limitations" in the Appendix for details).

Auto Keyframe ​

ParameterDescription
Auto Key FrameWhen turned ON, keyframes are automatically set whenever parameters are modified.

NOTE

Since keyframes are recorded with every move or rotation, you may end up with a large number of unintended keyframes or unintended animations. Keep this OFF if you are not animating.

Manual Keyframe Setting ​

ParameterDescription
Set Keyframes for Selected (Alt+K)Records the values at the current frame as keyframes for the currently selected characters/parts. Does not modify parameter values. (Shortcut key WinAlt+K/MacOption+K can also be used)

NOTE

The recorded scope depends on Selection Mode. All animatable parameters within that scope receive keyframes.

  • Global: The entire effect, including Global Settings, every character and part, the camera, and all effects
  • Character: All Character Settings for the selected characters, including Z Offset / Yaw / Pitch in 3D presets
  • Part: All Part Settings for the selected parts, including Z Offset in 3D presets

Non-animatable switches such as Use Indiv Color are not recorded.

Clean Up Redundant Keyframes ​

This feature scans all animated parameters and removes redundant keyframes whose values do not change. It provides standard and aggressive cleanup modes.

ParameterDescription
NEW Aggressive CleanupWhen OFF (default), standard cleanup is used. When ON, middle keyframes are also removed from runs of three or more consecutive keys with the same value.
Clean Up Redundant KeyframesOpens a confirmation dialog, then scans all animated parameters using the selected mode and removes redundant keyframes.
  • Aggressive Cleanup: OFF (default) — Only parameters whose keyframes all have the same value are cleaned. All keys on such a parameter are removed and replaced with one static value.
  • Aggressive Cleanup: ON — In addition to standard cleanup, when three or more consecutive keyframes have the same value, the middle keys are removed while the endpoints of the run are preserved.

CAUTION

An OFX plugin cannot inspect keyframe handles or interpolation curves. In aggressive mode, a same-valued middle key may still be important to the curve, but the plugin cannot detect that. Removing it may therefore change interpolation. Review the confirmation dialog and use this mode only when the host's Undo operation is available if needed.

NOTE

This is useful for cleaning up a large number of keyframes that may have been recorded unintentionally by "Auto Keyframe" during moves or transforms. This can help improve operation performance. The number of deleted keyframes will be notified in a dialog after completion.

Path Deformation Effects ​

In addition to the Hand Drawn effect, four deformation effects are available to distort character outlines in a regular, consistent way. Enabling (ON) any of these effects applies custom deformation and animation according to their parameters.

1. Pulse Effect ​

Expands and contracts the character outline outward and inward, creating a breathing-like motion.

2. Trapezoid Effect ​

Warps text into a trapezoid shape (perspective distortion) along a custom angle axis. By shifting the angle, you can modify the taper direction. Animating this parameter allows you to warp characters back and forth.

3. Sphere Effect ​

Projects the character outline onto a sphere, inflating (ballooning) or pinching it at the center.

4. Barrel Effect ​

Gradually changes the scale from the character's center toward the outside, creating a barrel (bulging) or pincushion distortion.

Hand Drawn Effect ​

Adds an organic, hand-drawn animation to the text by combining two effects: wobble, which distorts the outline paths of characters in a wave-like fashion, and jitter, which subtly shifts the position, rotation, and scale of each character frame by frame.

ParameterDescriptionDefault
EnabledTurns the effect on or off. Animatable.OFF
Pattern CountNumber of frames in the loop cycle — think of it like the number of cels in a flip-book animation. The effect cycles through this many distinct patterns before repeating.3
Wobble AmplitudeStrength of the path distortion (wobble). 0 disables wobble. Animatable.3.0
Fine WobbleStrength of the micro hand tremor (local wobble). 0 disables tremor. Animatable.0.15
Jitter AmplitudeStrength of the character position displacement (jitter). 0 disables jitter. Animatable.2.0
Frame RateHow many times per second the pattern switches. Lower values produce slower, smoother changes. Animatable.6.0
SeedSeed value that determines the random wobble/jitter pattern. Different seeds produce different patterns.0
TargetsSpecifies the characters/parts to apply the effect to. For the notation, see "Targets Format" (character unit c1 / part unit p5 / range c1-c3 / negation !c2, etc. are available). If empty, the effect is applied to all parts.(empty)

TIP

Tips for getting the most out of this effect

  • Simply turning Enabled ON gives you a natural hand-drawn wobble with the default values.
  • If the wobble feels too strong, lower Wobble Amplitude (outline ripple) and Jitter Amplitude (positional offset).
  • Adjust the speed of the wobble with Frame Rate. Lower values produce slower movement.
  • Setting Pattern Count to 1 freezes the wobble into a static (non-moving) hand-drawn look — handy when you want the texture without any motion.

Randomize ​

Simultaneously applies and keyframes random values within the specified range for selected characters/parts. Useful for quickly creating initial animation states.

ParameterDescriptionDefault Range
Pos XRandomize X position (ON/OFF + Min / Max)0.4 to 0.6
Pos YRandomize Y position (ON/OFF + Min / Max)0.4 to 0.6
ScaleRandomize Scale (ON/OFF + Min / Max)0.7 to 1.3
RotationRandomize Rotation (ON/OFF + Min / Max)-30° to +30°
Stretch XRandomize X Stretch (ON/OFF + Min / Max)0.8 to 1.2
Stretch YRandomize Y Stretch (ON/OFF + Min / Max)0.8 to 1.2
Randomize Selected (Set Key)Applies the checked items to the selected characters/parts and records keyframes.

NOTE

The values specified in Min / Max are absolute. They are not offsets to the current values, but the absolute values to be set directly.

Randomization Procedure:

  1. Turn ON the checkbox for the parameters you want to randomize.
  2. Set the Min / Max values.
  3. Select the target characters/parts on the preview screen.
  4. Press the Randomize Selected (Set Key) button.
